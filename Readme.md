jqpm_z
========

jqpm_z - jquery plugin manager

Installation
------------

	npm install jqpm_z


Usage
-----

addUser:

	jqpm addUser

    you can addUserinfo to jqpm_z
publish:

	jqpm publish <pluginName>[@version]

    publish your jquery plugin

install:

    jqpm install <pluginName>[@version]

    install jquery plugin
